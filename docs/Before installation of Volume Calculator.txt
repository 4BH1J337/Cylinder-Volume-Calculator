Volume calculator features the calculations of the various volume of geometric solids including a Cylinder, Cone, Cuboid and Sphere. Whether you want to figure how much water fits in the can or need to find the volume of a rectangular box.

The volume of every shape is calculated differently using different formulas. The input values can be given in multiple metric/unit:  mm, cm, in. The units  are in place to give an indication of the order of the result such as mm3,cm3 or Litres.

Application is built using Python programing language (3.10) and Tkinter framework for Graphical User Interface (GUI).
